# PFSystem

PFSystem is an experimental **Python-based market analysis and signal exploration framework**.

It is designed for:
- systematic observation of price behavior
- structural signal detection (e.g. Point & Figure logic)
- statistical and rule-based filtering
- post-market analysis workflows

PFSystem **does not provide financial advice** and is not a trading bot.
It is a research and learning system built entirely on public knowledge.
